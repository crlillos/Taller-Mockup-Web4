# Taller Mecanico - Proyecto de Servidor

Este es el servidor central del taller mecanico, programado con Python y el framework Flask.

## Tareas completadas en el Backend
- [x] Instalacion y configuracion inicial de Python y Flask.
- [x] Creacion de la estructura de carpetas para archivos estaticos y plantillas.
- [x] Configuracion de las rutas principales en app.py (Inicio, Login, Registro, Perfil, Reservas).
- [x] Creacion del archivo de dependencias (requirements.txt).
- [x] Documentacion tecnica inicial.

## Instrucciones para ejecutar el proyecto
Para que el servidor funcione en otro ordenador, se deben seguir estos pasos en la terminal:

1. Instalar las librerias necesarias:
pip install -r requirements.txt

2. Iniciar el servidor de Python:
python app.py

3. Acceder a la direccion local:
http://127.0.0.1:5000
